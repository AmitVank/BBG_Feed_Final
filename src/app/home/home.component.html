In today’s world data is considered as NEW oil, which comes from a myriad of different sources such as transaction systems, customer data, financial data, stock market data, various news channels, mobile application, IOT. Data is growing at an astronomical rate, companies, Individuals use data science, analytics and machine learning to name a few to sift through the goldmine of data to derive meaning full strategies. The outcome of mining the data using different science can anticipate and predict customer behavior, market trends to name a few patterns.

My project involves analyzing the stock prices to find crucial data points such as outliers. Outlier basically observes unusual patterns on the stock performance by using timeseries model. Time series models stocks prices correctly which helps the stock buyer decide when to buy or sell stocks. Time series looks at sequence of historical data to predict the future prices and behaviors. To accomplish this, I have built this website and the following tasks were performed.

Step 1: Angular CLI, Node.js, Git CLI and visual studio were downloaded and the environment was set up to build the website.
Step 2: The next important step was to identify the market data to analyze stock performance. Based on my research, I was able to identify free API from marketstack.com. Market stack API provided historical data stock prices from various exchanges around the world.
Step 3: Used Angular to build the front end webpage which calls the Marketstack using rest API
Step 4: The market data from the API was fed to Time series model
Step 5: Timeseries model analysis is done using the following Python packages and R in Jupyter notebook. The output of the timeseries model was displayed on the website

  • plotly - Used for interactive web-based visualizations that can be displayed in Jupyter notebooks
  • NumPy - Used to perform a wide variety of mathematical operations on arrays. It adds powerful data structures to Python that guarantee efficient calculations with arrays and matrices and it supplies an enormous library of high-level mathematical functions that operate on these arrays and matrices.
  • Pandas - Used to convert Numpy array to a Pandas data frame.
  • datetime - Used to work with date and time. These classes provide a number of functions to deal with dates, times and time intervals. Date and datetime are an object in Python, so when you manipulate them, you are actually manipulating objects and not string or timestamps.

Step 6: The website was hosted in Amazon web services using S3 bucket.
